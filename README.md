# Token Ring эмулятор

Эмулятор протокола Token Ring на Go.

## Запуск

```bash
go run main.go <число_узлов>
```

**Примеры:**
```bash
go run main.go 3
go run main.go 5
go run main.go 10
```

## Что делает

- Создает кольцо из N узлов
- Каждый узел - отдельная goroutine
- Узлы передают сообщения по кольцу
- Сообщения доставляются по SHA3 хэшу получателя
- TTL защищает от бесконечных циклов

## Требования

Go 1.24+ (для crypto/sha3)

## Установка Go 1.24

```bash
go install golang.org/dl/go1.24.0@latest
go1.24.0 download
go1.24.0 run main.go 5
```
